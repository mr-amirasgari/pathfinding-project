# Project Explanation

## Overview

This project demonstrates pathfinding in a 2D grid using the Breadth-First Search (BFS) algorithm. It simulates an agent navigating from a start point to a target while avoiding obstacles.

## Algorithm

- **BFS (Breadth-First Search)**: Guarantees the shortest path in an unweighted grid.

## Key Features

- Dynamic grid generation
- Random placement of start and goal
- Visual path display
- No physical movement during analysis

## Applications

- Game AI
- Robot navigation
- Route planning
- Simulation environments
